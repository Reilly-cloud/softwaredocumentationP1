Create an Interative PDF in Adobe InDesign

This help document will explain how to take a new or existing InDesign document and add interactive features including bookmarks, buttons, hyperlinks, page transitions and cross references. It will also cover how to preview and save the interactive features to a PDF document.

Click here for the table of contents.

Save the document as an Interactive PDF file

  • Choose File > Export.
  • Specify a name and location where to save the file.
  • For Save As choose Adobe Interactive PDF and then click save.
